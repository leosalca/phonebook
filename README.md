# Vue-Flask-MongoDB Phonebook Project
A Vue front-end and Flask back-end app. Integrated with USPS Web Tools API to validate zipcode

## Technologies used in the front-end
- JavaScript
- TypeScript
- Vue.js
  - Pinia
  - Vuelidate
  - Naive UI Components
- Node.js

## Technologies used in the back-end
- Python
- Flask
- MongoDB
